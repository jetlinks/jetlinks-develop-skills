# jetlinks-i18n

JetLinks internationalization skill for module-level i18n decisions, resource
files, `LocaleUtils`, and enum localization.

## Files

```text
jetlinks-i18n/
  SKILL.md
  README.md
  usage-guide.md
```

## What It Covers

- When a JetLinks module should add i18n at all
- `messages_zh.properties` and `messages_en.properties`
- `LocaleUtils` usage in synchronous and reactive code
- Errors, prompts, permissions, actions, enums, and user-facing text
- `I18nEnumDict` enum localization patterns
