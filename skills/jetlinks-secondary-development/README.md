# jetlinks-secondary-development

Self-contained OpenClaw skill for JetLinks-based scaffold secondary development.

## Structure

| Directory | Purpose |
|-----------|---------|
| `SKILL.md` | Main entry — task routing and constraints |
| `references/` | Rule files and the AI routing index |
| `agents/` | Agent configuration (OpenAI YAML) |
| `assets/` | Reserved for future use (images, templates, etc.) |
| `.claude/` | Claude Code IDE rules (optional) |
| `.cursor/` | Cursor IDE rules (optional) |
