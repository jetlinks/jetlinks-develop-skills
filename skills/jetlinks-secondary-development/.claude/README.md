# Claude Code 开发规则

本目录是 JetLinks 系脚手架在 Claude Code 下的入口说明。

## 使用方式

1. 先阅读 [../ai-prompt.md](../ai-prompt.md)
2. 根据任务类型读取对应的 `.prompt/*.md`
3. 始终以当前工作区现有代码为最终依据

## 设计目标

- 通用化：不依赖某个仓库的静态模块列表
- 轻量化：总路由只做分发，详细规则按需读取
- 一致化：`.claude`、`.codex`、`.cursor` 共用同一套核心规则

## 重要提醒

- 提示词只提供决策流程，不提供仓库快照
- 智能体必须先发现当前脚手架结构，再实现功能
